# flex 弹性布局盒模型
## 设置弹性盒模型
1. display:flex (新版)
2. display:-webkit-box; (旧版)
### flex容器
1. 项目排列方向
1.1 从左到右排练
1.2 row-reverse 从右到左
1.3 column从上到下
1.4 column-reverse从下到上

     flex-direction: row | row-reverse |column |column-reverse

horizontal 水平方向
vertical 垂直方向

-webkit-box-orient:horizontal | vertical

normal 顺序正常

reverse 反向

-webkit-box-direction:normal |reverse
